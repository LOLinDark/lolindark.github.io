Community Testing Program

Help validate OmniCore updates with a guided card-based routine designed for first-time users and contributors.

1. Open the Guide

Launch OmniCore and open the HOTAS Test Guide from the bookmark bar inside the app.

2. Run the Steps

Follow each test card from baseline setup through per-mode binding validation and regression checks.

3. Submit Feedback

Send one final report. Bugs and faults route to GitHub issues; pass and general feedback route through VerseMail.
